READ ME

See the scss file for URL to the template colour palette base.

Install on Mac with Homebrew: brew install sass/sass/sass


Command to convert the template.scss to css:
sass pmhc-mds.scss pmhc-mds.css
cp pmhc-mds.css ~/dev/pmhc-mds/pmhc-mds-app/app/static/css/


Note that the Bootstrap downloaded scss files must be in the root folder. 
Good advice from here: https://www.mugo.ca/blog/How-to-customize-Bootstrap-4-using-Sass
